# CRUD-Consultant
simple CRUD application that allows the creation of a consultant, reading his infos, modifying them, deleting them and consulting him

# Quick Start

First, clone the repository:
```
git clone git@github.com:0xjarix/consultant-crud-app.git
```

Then, enter the directory:
```
cd consultant-crud-app
```

Finally, run:
```
mvn spring-boot:run
```

Connect to `http://localhost:8080/` and play around.
